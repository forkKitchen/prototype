Meat Grandma

There are 9082 public libraries in the USA.Public libraries are local, trusted organizations, all of which can build unique programming opportunities around their unique food culture. Ultimately, the intention is to make ingredient-level recipe data available via an open API so that developers can build visualizations and other tools with it. We imagine fork.kitchen powering things like visualizations that describe how sweet bbq is by location across a region. By mashing up metadata made available through the DPLA API with recipes from fork.kitchen, narratives describing food and history can be traced across cultures, regions, and time.
